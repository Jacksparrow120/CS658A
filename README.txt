To run the codes:

Open jupyter notebook and run these 4 pynb files given:
1. feature_engineering.ipynb
2. project_model_on_engineered_data.ipynb
3. Project Bot-IoT.ipynb
4. project_model.ipynb

After doing so run all cells one by one. If any error comes due to package dependency, then install that package using " pip install {package_name} " command.
............................................................................................................
To run the frontend user interface, go to FRONT folder:
Type "streamlit run main.py" in the command prompt.
Note: streamlit package must be installed. If not, type: "pip install streamlit"
